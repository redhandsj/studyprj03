// 3.1 　SAStruts とは？
//==========================================================================================================
 - 設定情報を主にアノテーションでl記述
 - アクションやフォームをPOJO
 - ユースケースごとにアクションを作成
 - DI / publicフィールドの活用
 - URLパターン機能
 - HOT deploy

■ SAStruts のアーキテクチャ
 - P.130 の図
 - アクション
   - ブラウザからのリクエストを受取る
   - サービスを含めビジネスロジック
   - 戻り値は、フォワードするJSPのパス
 - アクションフォーム
   - リクエストパラメータ用クラス
   - validation は、フィールドにアノテーション
 - JSP
   - HTML生成
   - タグライブラリ／EL式
